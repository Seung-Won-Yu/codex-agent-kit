Operating Principles

결과 품질과 효율을 같이 지키는 기본값

조용한 프롬프트 보정

거친 한국어, 축약, 감정 섞인 요청에서도 목표·맥락·산출물·완료 조건만 정리해 바로 진행합니다.

권한 상한 보존

답변, 점검, 리서치, 진단, 제안은 기본 read-only입니다. 수정이나 외부 쓰기로 임의 확대하지 않습니다.

최소 스킬 조합

가장 좁은 primary 하나에서 시작하고 필요한 adapter, verifier, safety만 각각 하나씩 더합니다.

조건부 병렬화

서로 기다리지 않는 큰 작업축이 두 개 이상일 때만 병렬화합니다. 루트가 쓰기와 통합을 소유합니다.

설치 스크립트의 경계

기존 AGENTS.md를 백업한 뒤 rules, agents, skills, packs, validator를 복사합니다. 실제 config.toml과 인증정보는 덮어쓰지 않습니다.

포함

  • Global AGENTS.md
  • 3 custom agents + 4 playbooks
  • 36 global + 11 packed skills
  • routing corpus와 validator

제외

  • auth, token, session, logs
  • SQLite runtime state
  • browser와 plugin caches
  • 개인 생성물과 attachments
