Operating Shape

The current rule set prefers action over ceremony.

Clear Request

Inspect the relevant files, make the change, run focused verification, and report the useful result.

direct execution -> verification -> handoff

Fuzzy Request

Install

Review first, then copy the portable layer into ~/.codex.

git clone https://github.com/Seung-Won-Yu/codex-agent-kit.git
cd codex-agent-kit
./scripts/install.sh

# Preview pruning extra local skills
./scripts/prune-skills.sh

# Apply only after reviewing the dry run
./scripts/prune-skills.sh --apply
