Designed around intent

요청 해석부터 검증까지 하나의 운영 체계

더 많은 스킬을 켜는 대신, 매 순간 필요한 책임만 정확하게 연결합니다.

프롬프트 자동 보정

짧고 거칠거나 한국어·영어가 섞여도 목표, 필요한 맥락, 산출물과 완료 조건을 내부적으로 정리합니다.

권한을 넘지 않는 실행

확인·분석·제안은 read-only로 유지하고, 고쳐줘·적용해줘처럼 명확한 표현이 있을 때만 파일을 바꿉니다.

정확한 스킬 자동 연결

별도 router ceremony 없이 description matching으로 가장 좁은 primary를 고르고 필요한 도구만 조합합니다.

품질 중심 에이전트 사용

큰 독립 작업축만 병렬화하고, 루트가 쓰기·통합·최종 판단을 소유해 결과가 흩어지지 않게 합니다.

Routing Contract

필요한 역할만 최대 네 슬롯

스킬을 많이 켜는 것이 아니라 책임을 정확히 나누는 방식입니다.

Primary

내용과 결정을 소유합니다. 구현, 진단, 리서치, 설계 중 가장 정확한 스킬 하나를 선택합니다.

Adapter

산출물 형식을 담당합니다. DOCX, PDF, PPTX, spreadsheet처럼 파일 형식이 필요할 때만 추가합니다.

Verifier

결과를 독립 검증합니다. 테스트, 빌드, 브라우저 흐름이나 고위험 변경 리뷰가 필요할 때 붙입니다.

Safety

민감 경계를 보호합니다. 인증, 권한, 결제, webhook, production data 작업에만 조건부로 추가합니다.

10 Connected Plugins

스킬이 실제 도구와 파일 작업으로 이어지게

Documents

Word·DOCX 문서를 생성, 편집, redline하고 렌더링까지 검증합니다.

Spreadsheets

XLSX·CSV 분석과 workbook 생성을 담당합니다.

Presentations

PowerPoint·Google Slides용 발표 자료를 만들고 편집합니다.

PDF

PDF를 읽고 생성하며 페이지 렌더링으로 레이아웃을 확인합니다.

Template Creator

기존 문서를 재사용 가능한 Codex artifact template로 변환합니다.

Sites

웹사이트 source, version, production deployment와 access control을 관리합니다.

Browser

Codex 안의 독립 브라우저로 페이지 탐색, 입력과 화면 검증을 수행합니다.

Chrome

로그인 상태와 기존 탭이 필요한 작업을 사용자의 Chrome에서 처리합니다.

Computer Use

전용 API가 없는 macOS 앱과 데스크톱 UI를 직접 조작합니다.

Visualize

차트, 비교 도구, 시뮬레이터와 interactive visualization을 대화 안에서 만듭니다.

One-command Setup

저장소의 현재 상태를 그대로 설치

Codex desktop app과 CLI가 설치되고 로그인된 macOS 환경을 기준으로 합니다.

git clone https://github.com/Seung-Won-Yu/codex-agent-kit.git
cd codex-agent-kit
./scripts/install.sh --exact --with-config --plugins

한 번에 적용되는 구성

  • Global AGENTS.md와 quality-first config
  • 3 custom agents와 4 domain playbooks
  • 36 global + 11 packed personal skills
  • xhigh profile, validator, 10 plugins

기존 세팅도 되돌릴 수 있게

  • --exact는 기존 personal layer를 timestamp 백업
  • Codex 기본 .system skills는 그대로 유지
  • 프로젝트 경로만 manifest에서 맞춤
  • 설치 뒤 validator로 skill graph 확인
python3 "$HOME/.codex/scripts/validate-skills.py"
codex --profile xhigh
