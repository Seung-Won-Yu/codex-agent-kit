## Composition rules

- Primary: 작업 내용과 결정을 소유하는 정확한 스킬 한 개
- Adapter: DOCX, PDF, PPTX, spreadsheet 등 필수 산출물 형식 한 개
- Verifier: 독립 검증이 실제 결론을 바꿀 수 있을 때 한 개
- Safety: auth, 권한, 결제, webhook, production data 등 민감 경계에서 추가

단순 설명·작은 수정·명확한 명령은 스킬 없이 루트가 직접 처리합니다. `routing-doctor`는 Codex 설정 자체가 작업 대상일 때만 사용합니다.
